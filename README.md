### 👉 프로젝트 개요

- 장보고 어플리케이션은 장보기 상황에서 발생하는 불편함을 개선하기 위해 개발된 플랫폼이다. 소비자들에게 직접 지출 내역 관리, 필요한 양 이상의 불필요한 식자재 구입 등의 불편 사항을 설문조사로 받았고, 이를 토대로 여러 문제점들을 비교하고 분석하여 장을 볼 때 보조 역할을 할 수 있는 최적의 기능들을 선정하게 되었다.
